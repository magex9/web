Mobile Dataset Example

An example to show how lots of columns of data could be made mobile friendly.

Tables are great for showing tabular data and are semantically correct in many situations. Sometimes the data does not need to be groups in a tabular format but rather grouped by items defining a list of their attribute which could be styled to look like a table.

The following examples show how a section with information inside of it can be styled to work where the user can easily scroll left and right and also where the user has a narrow screen and want to scroll vertically, or print on different sizes of paper.
