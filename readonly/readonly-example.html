Readonly Form Example

An example to show how an editor fieldset could be showing using definition list tags

One of the challenges i have found in building web applications with a large amount of data entry forms is making printable copy of the forms. Many times text overflows outside of textareas and input fields, and only the selected options of mutliselect lists or checkbox lists should be shown.

In the past I have always used a large number of div tags with specific styling, however after reading an article on DL-DD-DT by Niels Matthijs, I realized there were different ways to use the >dl< tag then putting all definitions in a single list. In this example I have put each form field in its own definition list along with any other associated footnotes and messages about it.

Input Fields

Error: This should be your full name
<div class="form-group col-md-4">
 <label>Name: </label>
 <input name="name" type="text" class="form-control" value="Scott" />
 <div class="message label label-danger">Error: This should be your full name</div>
</div>

Div Containers

Name:
Scott
Error: This should be your full name
<div class="form-group col-md-4">
 <div class="field-label">Name: </div>
 <div class="field-value">Scott</div>
 <div class="field-message danger-message">Error: This should be your full name</div>
</div>

Definition List

Name:
Scott
Error:
This should be your full name
<dl class="name-field col-md-4">
 <dt class="definition-label">Name: </dt>
 <dd class="definition-value form-control">Scott</dd>
 <dt class="message-label message-danger">Error: </dt>
 <dd class="message-description message-danger">This should be your full name</dd>
</dl>
